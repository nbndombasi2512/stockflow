# M1-11 — Location create/edit/archive form

- **Milestone:** M1 — Foundations
- **Type:** Frontend
- **Estimate:** 1-2 hrs
- **Depends on:** M1-10

## Description

Build the React form (React Hook Form, styled-components) for creating and editing a location, plus an archive action. Present it in a Radix UI Dialog.

## Acceptance Criteria

- [ ] Create and edit modes share one React Hook Form.
- [ ] Validates required name field.
- [ ] Rendered in a Radix Dialog; archive uses a Radix AlertDialog confirmation.
- [ ] Submits via TanStack Query mutations and invalidates the locations query.
- [ ] Success/error feedback and return to list.
