# M4-07 — Low-stock indicators

- **Milestone:** M4 — Visibility
- **Type:** Frontend
- **Estimate:** 1 hr
- **Depends on:** M4-02

## Description

Add visual low-stock flags (React/styled-components) wherever products are listed (list, detail, dashboard).

## Acceptance Criteria

- [ ] Products at/below reorder threshold are visually flagged (Radix-based badge/indicator).
- [ ] Optional filter (feeding the TanStack Query key) to show only low-stock products.
- [ ] No flag shown when threshold is unset.
