# M0-01 — Scaffold React + TypeScript frontend

- **Milestone:** M0 — Project Setup
- **Type:** Setup / Frontend
- **Estimate:** 1-2 hrs
- **Depends on:** none

## Description

Initialize the frontend app using React + TypeScript bundled with Webpack and styled with styled-components. Install and wire the core libraries (React Hook Form, Radix UI, TanStack Query, TanStack Table). Wire up the existing `index.tsx` entry point and create a placeholder `App` component and routing shell.

## Acceptance Criteria

- [ ] React + TypeScript project builds via Webpack and runs locally.
- [ ] Webpack dev server with hot reload configured.
- [ ] styled-components configured (with a base theme/global style).
- [ ] TanStack Query `QueryClientProvider` wraps the app.
- [ ] React Hook Form, Radix UI, and TanStack Table installed and import-verified.
- [ ] `index.tsx` mounts an `App` component into `#root`.
- [ ] Client-side router installed with a placeholder route.
- [ ] `npm run dev` serves the app without errors.
