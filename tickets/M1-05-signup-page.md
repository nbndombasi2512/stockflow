# M1-05 — Signup page

- **Milestone:** M1 — Foundations
- **Type:** Frontend
- **Estimate:** 1-2 hrs
- **Depends on:** M0-01, M1-02

## Description

Build the signup screen as a React component styled with styled-components, using React Hook Form for the form and a TanStack Query mutation to call the signup endpoint.

## Acceptance Criteria

- [ ] Email + password fields managed by React Hook Form with validation.
- [ ] Submits via a TanStack Query mutation to `POST /auth/signup` and handles success/error states.
- [ ] Shows duplicate-email and validation errors.
- [ ] Redirects to the app on success.
