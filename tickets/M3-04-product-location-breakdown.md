# M3-04 — Per-location breakdown on product detail

- **Milestone:** M3 — Multi-location
- **Type:** Backend / Frontend
- **Estimate:** 1-2 hrs
- **Depends on:** M2-01

## Description

Show a product's quantity broken down by each location, plus total on hand.

## Acceptance Criteria

- [ ] NestJS endpoint (Prisma) returns per-location quantities for a product.
- [ ] Product detail displays the per-location breakdown (TanStack Table) and total, fetched via TanStack Query.
- [ ] Locations with zero stock handled clearly.
